Project Overview

Under construction: Ayurveda AI helps users discover and understand their unique mind-body constitution through the ancient wisdom of Ayurveda. Through a comprehensive quiz, personalized recommendations, and AI-generated recipes, the app guides users toward optimal health and balance in their daily lives.

Features

The app offers a complete Ayurvedic wellness experience:

  • Constitution Analysis:
    • In-depth 21-question dosha quiz
    • Detailed analysis of Vata, Pitta, and Kapha elements
    • Personalized constitution report
    • Progress tracking and reassessment tools
  • AI-Powered Recipe System:
    • Daily automated recipe generation via OpenAI ChatGPT API
    • 10 new dosha-specific recipes added every 24 hours
    • Recipe filtering by dosha type and meal category
    • Personal recipe collection with save functionality
  • Personalized Recommendations:
    • Dosha-specific dietary guidance
    • Customized lifestyle routines
    • Ayurvedic herb suggestions
    • Seasonal wellness adaptations
  • Educational Resources:
    • Comprehensive Ayurvedic principles
    • Mindfulness and breathing practices
    • Detailed articles and tips
    • Practical wellness guidance

Understanding Doshas

The app explores the three fundamental energies in Ayurveda:

  • Vata: Air and space elements, governing movement and change
  • Pitta: Fire and water elements, governing transformation and metabolism
  • Kapha: Earth and water elements, governing structure and stability

Technical Details

Built natively for iOS using SwiftUI, and for Android using Kotlin, the app features:

  • Core Features:
    • Intuitive quiz interface
    • Sophisticated dosha calculation algorithm
    • Local data storage for privacy
  • Content Generation System:
    • Automated Node.js script for recipe generation
    • Integration with OpenAI ChatGPT API
    • 24-hour content refresh cycle
    • MongoDB for recipe storage and management
  • Backend Infrastructure:
    • Express.js REST API
    • Automated content delivery system
    • Efficient recipe pagination and filtering
