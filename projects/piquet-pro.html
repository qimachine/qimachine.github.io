Piquet Pro

The classic 16th-century French card game on iOS. Traditional rules, strategic AI opponent, and a faithful two-phase gameplay system.

SwiftUI Game AI Card Game

Project Overview

Piquet Pro brings the sophisticated 16th-century French card game to iOS. Originally a favorite among European nobility, this digital adaptation maintains the game's elegant strategy while making it accessible to modern players — combining careful planning, tactical thinking, and just the right amount of chance.

Game Structure

  • Traditional 32-card deck — 7 through Ace, as per classic rules
  • Declaration phase — Point declarations, sequence announcements, and set combinations
  • Trick-taking phase — Strategic play for tricks and points
  • Comprehensive scoring — Full traditional scoring system faithfully implemented

Features

  • AI opponent — Strategic computer opponent with genuine decision-making logic
  • Intuitive touch interface — Card interactions designed for mobile play
  • In-game tutorial — Rules reference built into the app
  • Smooth card animations — Polished visual feedback throughout
  • Ad-free option — One-time purchase to remove banner ads

Technical Implementation

  • Platform: Native iOS
  • Framework: SwiftUI
  • AI: Custom game AI with strategic decision-making
  • State management: Efficient game state tracking across both phases
  • Monetization: AdMob banner ads with one-time removal purchase
