Piquet Pro

A timeless French card game for the modern era

Project Overview

Piquet Pro brings the sophisticated 16th-century French card game to iOS devices. Originally a favorite among European nobility, this digital adaptation maintains the game's elegant strategy while making it accessible to modern players. The game combines careful planning, tactical thinking, and just the right amount of chance to create an engaging experience.

Game Features

Experience this classic card game with modern conveniences:

  • Gameplay Elements:
    • Traditional 32-card deck (7 through Ace)
    • Complete two-phase gameplay system
    • Strategic declaration phase
    • Engaging trick-taking mechanics
  • Modern Features:
    • Intuitive touch interface
    • AI opponent with strategic gameplay
    • Comprehensive scoring system
    • In-game tutorial and rules reference

Game Structure

Each game of Piquet Pro follows the traditional format:

  • Card distribution and selection
  • Declaration phase:
    • Point declarations
    • Sequence announcements
    • Set combinations
  • Trick-taking phase
  • Final scoring and results

Technical Details

Built natively for iOS using SwiftUI, the app features:

  • Efficient game state management
  • Advanced AI decision making
  • Smooth card animations
  • Game progress tracking
  • Banner ad integration with removal option

Premium Features

While Piquet Pro is free to download and play, users can opt for an ad-free experience through a simple one-time purchase that removes all banner advertisements from the game.
