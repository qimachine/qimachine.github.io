Weather.exe

A fully functional weather app wrapped in a faithful Windows 95 aesthetic. Global coverage, no ads, no subscription — just the forecast.

SwiftUI Kotlin MVVM Open-Meteo API CoreLocation iOS Android

What it is

Weather.exe brings the Windows 95 UI to your phone's weather forecast. Every panel, title bar, and button is rebuilt from scratch in the iconic silver-gray aesthetic — beveled edges, raised panels, the classic teal desktop. It's a love letter to the golden age of desktop computing that also happens to give you an accurate, useful weather forecast.

The core differentiator is purely the interface. Weather data comes from Open-Meteo, which provides accurate global coverage with no API key or account required. The result is an app that's lightweight, fast, and free of third-party data entanglements.

Features

  • Current conditions — temperature, feels like, wind, precipitation, cloud cover
  • Hourly forecast — 48-hour scrollable outlook with icons and temperatures
  • 10-day forecast — daily high/low with weather icons
  • Detailed stats — wind speed, gusts & direction, sunrise/sunset, humidity, dew point, visibility, barometric pressure
  • Multiple saved locations — switch between any number of cities worldwide
  • Imperial & metric units — toggle between °F/°C, mph/km/h, inches/mm
  • Global coverage via Open-Meteo
  • No ads, no account, no subscription

Design approach

The entire UI is built from scratch in SwiftUI (iOS) and Kotlin (Android) to faithfully replicate the Win95 aesthetic — raised and sunken bevel effects, the classic teal desktop, silver dialog panels, and the unmistakable title bar blue. The goal was to make every screen feel like a genuine Win95 dialog box while remaining practical and easy to navigate.

No asset packs or shortcuts were used. Each bevel, shadow, and panel border is coded directly, giving full control over fidelity and performance.

Technical implementation

  • iOS built in SwiftUI with MVVM architecture and CoreData for saved locations
  • Android built natively in Kotlin
  • Weather data: Open-Meteo API (free, no account required)
  • Location: CoreLocation with reverse geocoding via MapKit
  • No third-party analytics or tracking libraries

Published on both the App Store and Google Play. Built, maintained, and updated by me — this is what I do when I build for myself.
