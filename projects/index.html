Software Projects

For over a decade, I've worked across the full spectrum of software development, from complex enterprise systems to innovative mobile applications. My experience spans the complete development lifecycle - from initial architectural decisions through deployment and maintenance. In recent years, I've embraced modern AI tools like ChatGPT and Claude to enhance both development efficiency and application capabilities. The projects below represent this diverse technical background. Each project presented unique technical challenges, from real-time audio processing in meditation apps to ensuring ADA compliance in web interfaces.

Many of these applications remain actively maintained, receiving regular updates to accommodate new mobile OS releases, security requirements, and user requests. This ongoing work provides valuable insights into building maintainable, scalable software that can evolve with changing technical requirements and user needs.
