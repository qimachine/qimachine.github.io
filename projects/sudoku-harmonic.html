Sudoku Harmonic

Sudoku meets sound therapy. Each number (1-9) plays a distinct Solfeggio healing frequency, turning every puzzle into a meditative experience.

SwiftUI Kotlin AVFoundation Audio Synthesis Core Haptics

Project Overview

Sudoku Harmonic transforms the classic number puzzle into a healing, meditative experience. Each number from 1 to 9 is mapped to a specific Solfeggio frequency (174 Hz - 963 Hz), creating an immersive sound therapy session as you play. Brain training meets nervous system regulation.

The Frequency Map

  • 1 → 174 Hz — Foundation and physical healing
  • 2 → 396 Hz — Liberation of fear and guilt
  • 3 → 417 Hz — Facilitation of change
  • 4 → 432 Hz — Universal harmony
  • 5 → 528 Hz — Transformation and DNA repair
  • 6 → 639 Hz — Relationship healing
  • 7 → 741 Hz — Awakening intuition
  • 8 → 852 Hz — Spiritual order
  • 9 → 963 Hz — Divine consciousness

Features

  • Real-time frequency synthesis — tones generated on-device, low latency
  • Haptic feedback — subtle vibration support for deeper immersion
  • Multiple difficulty levels — accessible to all skill ranges
  • Light and dark mode — clean, minimalist interface optimized for focus
  • Adjustable audio — control frequency volume and intensity independently
  • Headphone-optimized — designed for the best experience with earbuds or headphones

Technical Implementation

  • iOS: SwiftUI, AVFoundation (real-time frequency synthesis), Core Haptics, MVVM
  • Android: Kotlin, AudioTrack API for precise frequency generation, Vibrator API
  • Audio: Real-time Solfeggio synthesis (174-963 Hz) with smooth transitions and low-latency processing
  • Puzzle engine: Efficient generation algorithms with multiple difficulty balancing
