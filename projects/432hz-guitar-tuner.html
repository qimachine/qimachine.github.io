432 Hz Guitar Tuner

A precision chromatic tuner supporting 432 Hz, 440 Hz, and scientific pitch. For players who care about their tuning reference.

SwiftUI AVFoundation Audio MVVM

Project Overview

432 Hz Guitar Tuner is a professional-grade chromatic tuner designed for musicians who want both technical accuracy and flexibility in tuning standards. While optimized for guitar, the tuner works with any instrument and functions fully offline.

The app's core differentiator is its support for three distinct tuning references - presented in a clean, musician-first interface with clear educational context about each standard.

Tuning Standards

  • 440 Hz - The modern industry standard used in most contemporary music
  • 432 Hz (Verdi tuning) - A popular alternative tuning reference advocated by Giuseppe Verdi
  • 430.54 Hz Scientific Pitch - A mathematically derived system placing middle C at 256 Hz

Core Features

  • High-precision chromatic pitch detection (±1 cent accuracy)
  • Guitar-first interface with automatic string detection (E A D G B E)
  • Chromatic tuning support for any instrument
  • Low-latency, real-time visual feedback
  • Needle-style tuning display with color-coded accuracy indicators
  • Works completely offline - no account required
  • Dark mode, minimalist professional UI

Educational Content

The app includes a dedicated section explaining tuning standards in a factual, balanced way - covering the history of Verdi tuning, what scientific pitch is, and how modern standards evolved.

Technical Implementation

  • iOS: SwiftUI
  • Android: Kotlin, Jetpack Compose
  • Audio: AVFoundation with real-time pitch analysis
  • Architecture: MVVM
  • Performance: Battery-efficient, low-latency processing
  • Privacy: No unnecessary permissions or data collection
