432 Hz Guitar Tuner

A precision chromatic tuner supporting 432 Hz, 440 Hz, and scientific pitch

Project Overview

432 Hz Guitar Tuner is a professional-grade chromatic tuner designed for musicians who want both technical accuracy and flexibility in tuning standards. While optimized for guitar, the tuner works with any instrument and functions fully offline.

The app's core differentiator is its support for three distinct tuning references - standard modern tuning, alternative tuning, and scientific pitch - presented in a clean, musician-first interface with clear educational context.

Tuning Standards Supported

  • 440 Hz - The modern industry standard used in most contemporary music
  • 432 Hz (Verdi tuning) - A popular alternative tuning reference advocated by Giuseppe Verdi
  • 430.54 Hz Scientific Pitch - A mathematically derived system placing middle C at 256 Hz

Users can switch instantly between standards, with all pitch detection recalculated in real time.

Design Philosophy

The interface is intentionally minimal and professional. Subtle visual accents inspired by harmonic and geometric themes are used sparingly, ensuring the app feels like a serious tool first, with aesthetic character as a secondary enhancement.
