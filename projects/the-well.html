The Well App

Holistic wellness journey for body, mind, and being

Project Overview

The Well is a comprehensive wellness application developed in partnership with Max Wellness, offering both free and premium video courses designed to enhance holistic health. The app provides users with expert instruction across three key wellness dimensions: Body, Mind, and Being.

Features

The app includes a thoughtfully curated collection of wellness courses delivered through high-quality video content. Key features include:

  • Free introductory videos for all courses
  • Premium content library with complete course access
  • Three distinct wellness categories: Body, Mind, and Being
  • Intuitive user interface for course navigation
  • Ad-free viewing experience for subscribers
  • Regular content updates with new courses

Technical Details

The application was built using React Native, ensuring a consistent, high-quality experience across both iOS and Android platforms. The app features:

  • Efficient video streaming and playback
  • Secure subscription management
  • Cross-platform compatibility
  • In-app purchase integration
  • User progress tracking
