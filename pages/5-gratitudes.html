5 Gratitudes

Begin each day with mindful appreciation

Project Overview

5 Gratitudes encourages a deeper practice of gratitude by challenging users to record unique daily reflections. The app combines thoughtful features like personalized reminders and custom image sharing while maintaining complete privacy of your journal entries.

Key Features

  • Journaling:
    • Daily unique entry requirement for deeper reflection
    • Review previous entries by date
    • Local, private storage for all entries
    • Clean, focused interface
  • Sharing & Reminders:
    • Custom image generation for social sharing
    • Configurable daily notifications
    • Optional social media integration
    • Privacy-focused design

Mental Wellness Benefits

Regular gratitude practice through 5 Gratitudes can:

  • Build emotional resilience
  • Reduce negative thought patterns
  • Enhance mindfulness
  • Improve present-moment awareness
  • Create lasting positive habits

Technical Details

  • Native iOS implementation in SwiftUI
  • Secure local data storage
  • Custom image generation system
  • Notification management

Premium Features

The app is free to download and use, with an optional one-time purchase to remove banner advertisements for an undistracted journaling experience.
